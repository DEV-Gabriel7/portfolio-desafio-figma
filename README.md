## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio prático de Front-End, o objetivo era reconstruir uma interface visual a partir de um protótipo disponibilizado no Figma.

A proposta simulou um cenário comum no mercado de desenvolvimento web: transformar um design criado por um time de UI/UX em uma aplicação funcional utilizando tecnologias de marcação e estilização.

Foram aplicados conceitos de HTML semântico, organização estrutural, posicionamento de elementos e estilização com CSS, mantendo a máxima fidelidade visual em relação ao layout original.

O projeto é destinado principalmente para fins educacionais e demonstra a capacidade de interpretar protótipos, estruturar interfaces e reproduzir layouts profissionais utilizando tecnologias base da web.

## ✨ Funcionalidades

* ✅ Estruturação semântica com HTML5
* ✅ Reprodução fiel do protótipo fornecido
* ✅ Layout desenvolvido do zero
* ✅ Organização de conteúdo em seções
* ✅ Utilização de imagens e elementos visuais
* ✅ Estilização completa com CSS3
* ✅ Estrutura limpa e de fácil manutenção

## 🛠️ Tecnologias Utilizadas

### HTML5

Responsável pela organização estrutural da página, garantindo melhor semântica, legibilidade e manutenção do código.

### CSS3

Utilizado para estilização dos componentes visuais, espaçamentos, posicionamentos e reprodução do design proposto.

### Figma

Ferramenta utilizada como base para análise do layout e implementação da interface.
